# webrtc

SwordFishTheChordedPlayer1's WebRTC binary from [Axel Isouard](https://github.com/aisouard)'s [libwebrtc](https://github.com/aisouard/libwebrtc) -- Prebuilt version

This repository contains the SwordFishTheChordedPlayer1's WebRTC Static Binary, recommended to use in your C++ projects.
As a part of [Axel Isouard](https://github.com/aisouard)'s [libwebrtc](https://github.com/aisouard/libwebrtc), we share WebRTC for Windows based on version M57 by SwordFishTheChordedPlayer1.

Download the SFTCP1's WebRTC Binary for Windows and other platforms from [Releases](https://github.com/SFTCP1/webrtc/releases) here (Compatible with Visual Studio 2015 or higher for Windows systems [x86 and x64]).
